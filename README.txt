Clipboard and clients are dependent on the library protobuffers to handle the serialization of message.
Before compiling any program run: make install
If that does not work visit:
1 - Visit https://github.com/google/protobuf and install protobuf interpreter
2 - Visit https://github.com/protobuf-c/protobuf-c and install protobuffers c wrapper

After that everything should be ready to compile the code: make all
